# Python Scientific Calculator

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Sine
- Cosine
- Tangent
- Inverse (1/x)

## Technologies Used
- Python
- math module

## How to Run

1. Clone the repository:
   git clone https://github.com/your-username/Python-calculator.git

2. Navigate to folder:
   cd Python-calculator

3. Run:
   python calculator.py
